Some important details

Working on a specific region

In the first examples, the positions are given by coordinates and no network is specified.

However when no coordinates are provided, you need to provide on what region you want to request.

The list of region covered by navitia are available at http://api.navitia.io/v0/regions.json . The shape of the region is provided in GeoJSON.

More about entry points

In the first example, we used coordinates. If you want to use any other point (e.g. a stop point), just use its uri. The uri of the different objects are returned by APIs as proximity list.

You should not try to build an uri by yourself, but only re-use those provided by an API. The only exceptions are coordinates that have the following syntax coord:lon:lat.
